Voice Assistant Widget Embed Demo

This page demonstrates how to embed the voice assistant widget in any website. The voice assistant appears below and can be interacted with directly.

Live Demo

How to Embed

Add the following code to your HTML:

<!-- Add React dependencies -->
<script src="https://cdn.jsdelivr.net/npm/react@18.3.1/umd/react.production.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/react-dom@18.3.1/umd/react-dom.production.min.js"></script>

<!-- Add widget script -->
<script src="https://your-domain.com/widget/voice-assistant-widget.umd.js" defer></script>

<!-- Add widget element -->
<voice-assistant agent-id="YOUR_AGENT_ID"></voice-assistant>

Attributes

  • agent-id - Your ElevenLabs agent ID (default: bk6zd2EX7Rc7o9LnxEtY)
